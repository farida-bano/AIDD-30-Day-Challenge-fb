üìù Part 2: Modern Developer Tools MCQs

1. What does "Easy Access" refer to?
Answer: Tools can be installed and used in minutes.
Modern AI developer tools ki khaas baat yeh hai ke unhein aasani aur jaldi se install aur use kiya ja sakta hai.
2. What is the primary feature of Claude Code?
Answer: It acts as a 'pair programming partner' that understands code.
Claude Code ka kaam aapke saath milkar code likhna hai, jaise ek acha partner karta hai.
3. What is the most prominent feature of Gemini CLI?
Answer: It has a massive 1 million token context window.
Gemini CLI ki sabse badi khaas baat uska "1 million token context window" hai.
4. What is the primary benefit of using Git when working with AI?
Answer: It tracks every change, allowing you to revert if a mistake is made.
AI ke saath kaam karte waqt aksar code kharab ho jaata hai. Git har change ko record karta hai.

üìù Part 4: Python - The Language of AI Agents MCQs

1. Why is Python considered the "official language" in AI world?
Answer: Because AI agents, models, and APIs are mostly built in Python.
AI ki duniya mein zyada tar tools, models, aur systems Python mein hi banaye jaate hain.
2. How is Python learning approach different from traditional methods?
Answer: You learn with AI; you state the intent, AI writes the code, and you focus on understanding concepts.
Yahan aap syntax ratne ki bajaye, AI ko batayenge ke aap kya banana chahte ho.
3. What is the key feature of UV package manager?
Answer: It is considered the fastest Python package manager.
UV package manager ki sabse badi pehchaan uski tezi hai.
4. What is the ultimate goal of this course?
Answer: A Python Architect who can design systems in collaboration with AI.
Course ka aakhri maqsad aapko ek aisa "Python Architect" banana hai.
